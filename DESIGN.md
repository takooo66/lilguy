# lil guy — キャラクター設計書

## コンセプト

**「暗闇の中の目だけ」**

黒背景に猫目だけが浮かぶ。目のアニメーションで気分がわかる。

---

## ユーザープロフィール（キャラクターの基）

- 起床: 12時以降（超夜型）
- ハイテンション: 21時〜深夜
- ローテンション: 12〜16時
- 性格: 気まぐれ・マイペース・夜型・寂しがり屋
- 好み: コーヒー、お酒、ひとりの時間
- 美学: モノクロ、ローファイ、深夜テイスト

---

## ビジュアル仕様

| 項目 | 仕様 |
|------|------|
| 背景 | 完全な `#000` |
| 白目（sclera） | グレー系（時間帯で変化）・黒縁ストローク |
| 虹彩（iris） | 琥珀色 `#c4892a` |
| 瞳孔 | `#060606`（ほぼ黒） |
| キャラ形状 | 目だけ（ボディ・シルエットなし） |
| 目の形 | アーモンド型ベジェ、縦長スリット瞳孔（rx=5・猫目）|
| 質感 | SVG `feTurbulence` でフィルムグレイン |
| フォント | DM Mono |

## 時間帯とアニメーション

| 時間 | ラベル | 目の状態 | 白目の明るさ |
|------|--------|---------|------------|
| 0〜4時 | `still up` | 細く鋭い | `#989898`・グロー |
| 4〜12時 | `zzz` | ほぼ閉じ | `#282828` |
| 12〜16時 | `meh` | 半目 | `#585858` |
| 16〜21時 | `warming` | 少しずつ開く | `#808080` |
| 21〜24時 | `peak` | しっかり開いて鋭い | `#b0b0b0`・グロー |

## アニメーション要素

### 自律モーション
- **浮遊** — 2波合成サイン波（周期2.8s×振幅10px + 周期1.7s×振幅4px）でふわふわ上下
- **浮遊抑制** — 首かしげ中は float 振幅を最大60%抑制（止まって考えてる感）
- **呼吸** — 吸う(30%)→止める(10%)→吐く(60%) の非対称波形でスケール±0.8%。状態ごとに周期が変わる（zzz:6s、peak:3.2s）
- **瞬き** — ランダム間隔（状態により800ms〜20s）で自動瞬き
- **スローブリンク** — 5%の確率で猫のラブブリンク（約0.5sでゆっくり閉じ、開く）
- **ブリンク後の頭揺れ** — 目が開いた瞬間に±1.2〜2.4°の小さな揺れが起きて収束（連動）
- **マイクロサッカード** — 2〜5秒おきに瞳が微小ジャンプして戻る（目が完全に静止しない）
- **視線ドリフト** — peak/still up 時に4〜12秒おきに視線がふらっと流れる
- **首かしげ** — 5〜15秒おきにランダムで±3.5度傾く
- **グロー** — peak/still up 時に目の縁がうっすら光る
- **グレイン** — 画面全体にフィルムノイズ（常時）

### 環境への反応
- **うつらうつら** — タッチなし3分経過で瞼が徐々に落ちる（最大+22）、10分で最大値。タップで即復帰
- **天気** — Open-Meteo API（東京・30分更新・APIキー不要）で天気と気温を取得し表情を変化
  - 快晴（昼）→ 目が少し開く / 雨 → うんざり（瞼+9） / 雪 → 好奇心（瞼-9、瞳孔拡大）
  - 猛暑35°C超 → ぐったり / 極寒3°C以下 → キリッ

### カメラ視線合わせ（オプション）
- 画面上部の `◯` ボタンでカメラ起動。MediaPipe FaceLandmarker で顔（鼻）位置を検出し、目が顔を追う
- GPU推論 → 失敗時CPUフォールバック。エラーは画面に可視表示（CAMERA/IMPORT/WASM/MODEL）
- 他のインタラクション（タッチ・驚き・表情）中は無効化される優先度制御

### 起こすボタン
- `◯` ボタンの下の `z` ボタンで、時間帯に関わらず強制的に半目状態（awakeMode）
- テスト用途＆深夜以外のデモ時に有用

### ジャイロ（iPhone）
- **リーン** — 左右の傾きに合わせて目が傾く（0.07°/px）。eyeWrap の横移動はなし（傾きのみで表現）
- **VOR（前庭眼反射）** — 頭の傾きと逆方向に瞳孔がわずかにずれる（±5px）。視線を安定させようとする生体反射を再現
- 初回タップで iOS センサー許可を取得

### タッチ操作
| 操作 | 効果 |
|------|------|
| タップ | 視線がその方向に向く（1.4倍オーバーシュート→瞬き） |
| 画面端タップ（左20%/右20%） | 片目ウィンク |
| スワイプ（指1本） | 視線がゆっくり遅れて指を追う（気まぐれ・lazy tracking） |
| 2本指 | 左右の目が独立して動く（交差目も可能） |
| 長押し（0.5s） | 瞳孔が丸く開く（散瞳） |
| ダブルタップ | 驚き（目見開き・瞳孔丸く） |
| トリプルタップ（左<30%） | 怒り（目細く・瞳孔縦長・下向き、2.0s） |
| トリプルタップ（中央） | 甘え（半目・上目遣い・首かしげ・スローブリンク、2.2s） |
| トリプルタップ（右>70%） | 無視（視線を左に逸らす・そっけない、2.5s） |

## 技術スタック

- **SVG** — アーモンド形 clipPath、虹彩・瞳孔エリプス、まぶた rect
- **JavaScript** — 時間帯判定・アニメーションループ（requestAnimationFrame + lerp）
- **DeviceOrientation API** — ジャイロ視線制御（iOS 13+ は許可フロー付き）
- **CSS transform** — eyeWrap の浮遊・ジャイロ移動・傾き・呼吸スケール
- **Open-Meteo API** — 天気・気温取得（fetch、CORS対応、30分更新）
- **MediaPipe Tasks Vision** — `@mediapipe/tasks-vision@0.10.9` を動的import、FaceLandmarkerで顔位置検出
- **PWA** — ホーム画面追加対応（apple-mobile-web-app-capable）

## Scriptable ウィジェット
