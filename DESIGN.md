# lil guy — キャラクター設計書

## コンセプト

**「暗闇の中の目だけ」**

黒背景に猫目だけが浮かぶ。目のアニメーションで気分がわかる。

---

## ユーザープロフィール（キャラクターの基）

- 起床: 12時以降（超夜型）
- ハイテンション: 21時〜深夜
- ローテンション: 12〜16時
- 性格: 気まぐれ・マイペース・夜型・寂しがり屋
- 好み: コーヒー、お酒、ひとりの時間
- 美学: モノクロ、ローファイ、深夜テイスト

---

## ビジュアル仕様

| 項目 | 仕様 |
|------|------|
| 背景 | 完全な `#000` |
| 色 | 白・グレー・黒のみ（モノクロ徹底） |
| キャラ形状 | 目だけ（ボディ・シルエットなし） |
| 目の形 | アーモンド型、縦長瞳孔（猫目） |
| 質感 | SVG `feTurbulence` でフィルムグレイン |
| フォント | DM Mono |

---

## 時間帯とアニメーション

| 時間 | ラベル | 目の状態 | アニメーション |
|------|--------|---------|--------------|
| 0〜4時 | `still up` | 細く鋭い | ゆっくり瞬き、薄くグロー |
| 4〜12時 | `zzz` | 閉じてる（弧の線のみ） | ときどき半分開く |
| 12〜16時 | `meh` | 半目 | ぼーっとした動き、瞳孔小さい |
| 16〜21時 | `warming up` | 少しずつ開く | 瞳孔が広がるアニメ |
| 21〜24時 | `peak hours` | しっかり開いて鋭い | 瞳孔くっきり、視線が左右に動く |

---

## アニメーション要素

- **瞬き** — ランダム間隔でゆっくり閉じて開く
- **視線移動** — 瞳孔がわずかに左右に動く（peak hours 時）
- **グロー** — 目の縁がうっすら光る（深夜ほど強い）
- **グレイン** — 画面全体にフィルムノイズ（常時）
- **開き具合** — 時間帯によってまぶたの高さが変わる

---

## 技術スタック

- SVG — 目の形状描画
- CSS アニメーション — 瞬き・視線・グロー
- JavaScript — 時間帯判定 → 目の状態切り替え
